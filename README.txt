Introduction:
This is a Python-based "Connect 4" game where a single player competes against an AI opponent. The game features a graphical user interface built with Pygame and AI decision-making using the Minimax algorithm.
---------------------------------------------------------------------------------------------------------------------------------
Features:
Interactive GUI using Pygame.
AI opponent with strategic decision-making.
Save and load game functionality.
Detects winning conditions for horizontal, vertical, and diagonal patterns.
Turn-based gameplay with visual updates.
---------------------------------------------------------------------------------------------------------------------------------
Ensure you have Python 3.9+ installed.
Install Pygame using the command in a terminal: 

pip install pygame


Download or clone this project.
Run the game using a terminal: 

python connect4.py
---------------------------------------------------------------------------------------------------------------------------------
How to play:

Run the game and enter your name.
Choose a column to place your piece by clicking it.
The AI opponent will then make its move in a timespan of 0.9 seconds.
Win by aligning 4 pieces horizontally, vertically, or diagonally before the AI does.
Save and load options are available for convenience.
---------------------------------------------------------------------------------------------------------------------------------
AI Implementation:
The AI uses the Minimax algorithm, which evaluates all possible outcomes up to a specific depth and selects the optimal move by minimizing the opponent's advantage and maximizing its own chances of winning.
